Path | Type | Description
---- | ---- | -----------
`error` | `String` | The HTTP error that occurred, e.g. `Invalid field`
`message` | `String` | A description of the cause of the error
`reasons` | `Null` | The list of reasons which caused error
`path` | `String` | The path to which the request was made
`status` | `Number` | The HTTP status code, e.g. `400`
`timestamp` | `String` | The time at which the error occurred
